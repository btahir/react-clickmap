# Security Policy

## Supported Versions

Security fixes are prioritized for the latest published minor release.

## Reporting a Vulnerability

Please report vulnerabilities privately to `security@bilaltahir.dev`.

When reporting, include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any proof-of-concept details

Please do not open public issues for security vulnerabilities. We will
acknowledge receipt and provide status updates as quickly as possible.

## Disclosure Process

- We confirm receipt within 72 hours.
- We triage and validate the report.
- We prepare and ship a fix.
- We coordinate disclosure after a patch is available.
